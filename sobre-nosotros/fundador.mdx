ServiTec Pmá fue fundada por Felipe Didier Santos Sanjur, técnico y estudiante de Ingeniería Electromecánica con formación en sistemas eléctricos, programación, desarrollo web y automatización.

## Formación académica

Felipe cursa la carrera de **Ingeniería Electromecánica**, disciplina que integra principios de ingeniería eléctrica y mecánica para el diseño, mantenimiento y diagnóstico de sistemas industriales y de automatización.

Esta base académica es complementada con práctica directa en proyectos reales de software, desarrollo web y sistemas de control lógico.

## Proyectos y experiencia

Felipe ha desarrollado proyectos que combinan múltiples disciplinas técnicas:

- **Sistemas de software**: aplicaciones funcionales con lógica de negocio, validación de datos y gestión de estados
- **Interfaces interactivas**: páginas y formularios web con comportamiento dinámico usando HTML, CSS y JavaScript
- **Formularios conectados a bases de datos**: sistemas donde los datos ingresados por usuarios se almacenan y gestionan en bases SQL
- **Paneles de administración**: interfaces internas para gestionar información de clientes, solicitudes y registros de servicio
- **Aplicaciones de control lógico**: sistemas que automatizan decisiones o procesos secuenciales basados en condiciones específicas

## Filosofía de trabajo

El enfoque de Felipe es **analítico, estructurado y orientado a la calidad**. Antes de proponer una solución, analiza el problema. Antes de ejecutar, define el alcance. Y antes de entregar, verifica que funciona.

Esta forma de trabajar es la base de cómo ServiTec opera: diagnóstico primero, ejecución después.

## Visión para ServiTec

ServiTec nació como un proyecto individual con foco en servicios técnicos directos. La visión de Felipe es convertirla en una **empresa técnica integral** capaz de atender proyectos complejos que requieren integración de sistemas eléctricos, mecánicos y digitales bajo un solo equipo coordinado.

El objetivo no es crecer por crecer — es mantener la calidad del diagnóstico individual a mayor escala, con más técnicos formados bajo los mismos estándares.
