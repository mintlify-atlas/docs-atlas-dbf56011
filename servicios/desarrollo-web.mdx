## Tecnologías que utilizamos

En ServiTec Pmá utilizamos tecnologías modernas y probadas para construir sitios web rápidos, seguros y mantenibles:

| Tecnología | Uso |  
|---|---|
| **HTML / CSS** | Estructura y diseño de páginas web |
| **JavaScript** | Interactividad, validaciones y lógica del lado del cliente |
| **Cloudflare Workers** | Backend sin servidor para formularios, APIs y lógica de negocio |
| **Cloudflare D1** | Base de datos relacional para almacenamiento de solicitudes y registros |
| **Cloudflare Pages** | Despliegue y hosting del sitio web con CDN global |

<Info>
  Cloudflare Workers y D1 nos permiten desplegar sitios con backend funcional sin los costos de servidores dedicados. Esto significa que los clientes obtienen una solución web profesional a un precio accesible, con infraestructura de nivel empresarial.
</Info>

## ¿Qué tipo de negocio puede solicitar este servicio?

La División de Desarrollo Web de ServiTec atiende:

- **Negocios pequeños y medianos** que necesitan su primera presencia web profesional
- **Profesionales independientes** (médicos, abogados, consultores, técnicos) que quieren mostrar sus servicios en línea
- **Empresas con procesos internos** que se gestionan en papel o en hojas de cálculo y quieren digitalizarlos
- **Emprendedores** que lanzan un producto o servicio y necesitan una landing page para su campaña

## ¿Qué incluye el proceso de desarrollo?

<Steps>
  <Step title="Evaluación de requerimientos">
    Se revisan las necesidades del cliente: tipo de sitio, secciones requeridas, funcionalidades, identidad visual existente y plazo esperado. Esta etapa puede realizarse de forma presencial o remota y tiene un costo de $20.
  </Step>
  <Step title="Cotización y propuesta">
    Con base en los requerimientos, se elabora una propuesta con el alcance del proyecto, las funcionalidades incluidas, el tiempo de entrega estimado y el costo total.
  </Step>
  <Step title="Desarrollo">
    Se construye el sitio web con las especificaciones acordadas. El cliente recibe avances para revisión y puede solicitar ajustes dentro del alcance pactado.
  </Step>
  <Step title="Entrega y publicación">
    Se realiza la publicación del sitio en el dominio del cliente, se verifica el funcionamiento completo y se entrega el código fuente. Se incluye soporte post-lanzamiento para correcciones menores.
  </Step>
</Steps>
