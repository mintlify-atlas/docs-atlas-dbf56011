Con la cotización aprobada, el equipo de ServiTec procede a la ejecución del trabajo. Esta fase cubre desde la preparación del área hasta la entrega final al cliente, incluyendo la orientación de mantenimiento.

## Ejecución profesional

Todo trabajo se realiza bajo normas técnicas establecidas para cada disciplina. La seguridad es la primera prioridad: no se ejecuta ningún trabajo que represente riesgo no controlado para personas, equipos o instalaciones.

Durante la ejecución, el técnico:

- Trabaja con herramientas y materiales apropiados para cada tarea
- Aplica procedimientos de seguridad específicos según el tipo de trabajo (eléctrico, mecánico, lógico)
- Documenta el avance y cualquier hallazgo adicional que surja durante el trabajo
- Notifica al cliente si aparece un problema fuera del alcance cotizado antes de proceder

## Proceso de trabajo

<Steps>
  <Step title="Preparación del área">
    El técnico verifica que el área de trabajo cumpla con las condiciones necesarias para operar con seguridad. Esto incluye corte de energía cuando corresponde, despeje del espacio y verificación de herramientas y materiales disponibles.
  </Step>
  <Step title="Ejecución técnica">
    Se realiza el trabajo según el alcance definido en la cotización aprobada. El técnico aplica las normas técnicas vigentes para la disciplina correspondiente — eléctrica, mecánica, programación o desarrollo.
  </Step>
  <Step title="Verificación de calidad">
    Una vez completado el trabajo, el técnico realiza pruebas de funcionamiento para confirmar que el problema fue resuelto y que el sistema opera correctamente dentro de parámetros normales.
  </Step>
  <Step title="Entrega al cliente">
    El técnico presenta los resultados al cliente: qué se hizo, qué se encontró y cómo quedó el sistema. Se entrega un resumen del trabajo realizado y se documenta la conformidad del cliente.
  </Step>
  <Step title="Orientación de mantenimiento">
    Al finalizar, el técnico orienta al cliente sobre el uso correcto del sistema reparado o instalado, señales de alerta a observar y recomendaciones de mantenimiento preventivo para extender la vida útil.
  </Step>
</Steps>

## Entrega y verificación final

La entrega no es simplemente el momento en que el técnico se retira. Implica una verificación conjunta con el cliente de que el trabajo cumple con lo cotizado y que el sistema funciona según lo esperado.

El cliente tiene la oportunidad de hacer preguntas, solicitar explicaciones y confirmar su conformidad antes de cerrar el servicio.

<Note>
  ServiTec hace seguimiento post-servicio para verificar que el trabajo mantuvo su resultado. Si se presenta algún inconveniente directamente relacionado con el trabajo ejecutado dentro del período de seguimiento acordado, el técnico evalúa la situación sin costo adicional.
</Note>
